# customer-behaviour-prediction
ML project to predict customer purchase behaviour using Logistic Regression, Random Forest, and SVM with Tkinter deployment.
# Customer Behaviour Prediction

This project predicts whether a customer will purchase a product based on Gender, Age, and Salary.

## Models Used
- Logistic Regression
- Random Forest
- SVM (Best Model)

## Accuracy
~90%

## Features
- Data preprocessing
- Visualization
- Model comparison
- Tkinter GUI for prediction

## Files
- project.ipynb
- Customer_Behaviour.csv
